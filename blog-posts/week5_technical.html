The Map Method

 

Similar to .each method, the .map method runs each element of the array through the code block and returns the processed elements.

Unlike the .each method, the .map method will actually return a new array of the altered elements. This new array is the same size as the original, in the same order, but with the new elements.


Implementing the map method is pretty easy stuff. Take the name of your array and slap .map behind and youâ€™re gravy.

 

Example:

#Our Array:

arr = [ 1, 2, 3, 4, 5]

 

#daisy chained with another defined method:

arr.map.reverse

 

#returns:

[ 5, 4, 3, 2, 1 ]

 

#With a method:

arr.map {|num| num * 2}

 

#returns:

[ 2, 4, 6, 8, 10 ]
