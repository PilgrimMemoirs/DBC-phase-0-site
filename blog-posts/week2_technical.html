Inline vs Inline Blocks


September 07, 2014

Make the most of Navigations and link lists
by knowing the difference between using blocks and inline blocks.



I'm going to explain the difference between using blocks vs inline-blocks with using navigations or other list of links as an example - they can be used for a slew other purposes, but I find this information is the most crucial when it comes to organizing your sites navigation.

If you prefer your links to be displayed vertically and stacked ontop of one another, you would use display: block; in your css. Blocks force a line break, hence being stacked horizontally. From there you have the option of selecting the height and width, margins, ect.

For most naviagtions, diplaying the links as blocks that are laid out horizontally typically looks and fits nicer on a page. That is where inline-block comes in. By using display: inline-block; in your CSS, the links will line-up next to each other horizontally. From there you can style the block with margins, heights and widths.
